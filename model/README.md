## To setup venv for jupyter notebook
```
# Create and active virtual env
python3 -m venv venv
. venv/bin/activate

# Install kernel
python3 -m ipykernel install --user --name=hello-nano

# In vscode, add the venv full path to Settings -> Venv Path
# Restart vscode
# Afterwards Select Kernel list should show the venv

```
